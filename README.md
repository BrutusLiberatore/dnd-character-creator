# D&D Character Creator

Applicazione web per creare personaggi di Dungeons & Dragons 5a Edizione.

## Funzionalità

- Scelta di nome, razza, classe, sottoclasse e background
- Calcolo automatico dell'equipaggiamento base
- Generazione artwork in pixel art tramite AI

## Tecnologie

- **Frontend:** React con Vite
- **Backend:** Java con Spring Boot (in sviluppo)
- **Database:** PostgreSQL (in sviluppo)

## Come avviare il frontend

1. Entrare nella cartella frontend: `cd frontend`
2. Installare le dipendenze: `npm install`
3. Avviare il server di sviluppo: `npm run dev`
4. Aprire il browser su `http://localhost:5173`
